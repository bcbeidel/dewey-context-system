# Competitive Analysis Template

**Purpose**: Structured template for systematic competitive intelligence gathering and analysis.

**Version**: 1.0
**Created**: 2026-02-08

---

## How to Use This Template

1. **Copy template** for each competitor
2. **Fill in sections** systematically
3. **Update quarterly** (or when significant changes)
4. **Compare across competitors** using comparison matrix
5. **Store in intelligence hub** for team access

## Monitoring & Updates

### Update Frequency

**Quarterly** (every 3 months):
- Review all competitor profiles
- Update pricing and features
- Check customer sentiment (reviews)
- Update SWOT

**Monthly**:
- Monitor news and announcements
- Check job postings (hiring trends)
- Social media sentiment

**Real-time** (set up alerts):
- Major announcements (funding, acquisition, launch)
- Executive changes
- Customer wins/losses
- Pricing changes

### Alert Setup

**Google Alerts**:
- "[Competitor name] announces"
- "[Competitor name] funding"
- "[Competitor name] customers"

**Social media monitoring**:
- Twitter lists for competitor accounts
- LinkedIn company page follows

**Review sites**:
- G2 email alerts for new reviews
- Capterra review notifications

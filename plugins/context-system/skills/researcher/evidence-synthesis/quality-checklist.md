# Systematic Review: Quality Checklist

**Purpose**: Comprehensive quality checklist for PRISMA 2020 compliance and quality assessment.

---

## PRISMA 2020 Compliance

Before finalizing literature review, complete all checklists:

### Core Requirements
- [ ] **Protocol**: PRISMA-P 17-item checklist complete → [[protocol-phase]]
- [ ] **Registration**: PROSPERO registration number provided
- [ ] **PRISMA 2020**: 27-item main checklist complete → [[prisma-compliance]]
- [ ] **Abstract**: 13-item abstract checklist complete
- [ ] **Flow Diagram**: PRISMA flow diagram included

### Quality Assessment
- [ ] **AMSTAR-2**: Self-assessment complete (target: High/Moderate confidence) → [[quality-assessment]]
- [ ] **ROBIS**: Risk of bias assessment complete (target: Low risk)
- [ ] **Sensitivity analyses**: Robustness checks conducted
- [ ] **Publication bias**: Assessed if meta-analysis conducted

### Content Requirements
- [ ] Research question clearly stated (PICO framework)
- [ ] Inclusion/exclusion criteria documented
- [ ] Full search strategy provided (all databases)
- [ ] Duplicate screening/extraction reported (Cohen's kappa)
- [ ] Risk of bias assessed for all studies (tool specified)
- [ ] All papers have proper citations
- [ ] Methodology comparison table included
- [ ] Findings organized thematically (not paper-by-paper)
- [ ] Contradictions acknowledged and analyzed
- [ ] Heterogeneity investigated (if meta-analysis)
- [ ] Research gaps identified
- [ ] Theoretical frameworks discussed
- [ ] GRADE certainty assessment (if applicable)
- [ ] Implications section present (practice, policy, research)
- [ ] Conflicts of interest declared
- [ ] Full reference list in consistent format

---

## Quick Pre-Submission Check

**Protocol Phase** (before starting):
- Research question defined
- PROSPERO registration complete
- Search strategy documented

**Data Collection Phase**:
- All databases searched
- Screening done by 2+ reviewers
- Data extraction standardized

**Analysis Phase**:
- Quality assessment completed
- Risk of bias evaluated
- Synthesis strategy appropriate

**Reporting Phase**:
- PRISMA 2020 checklist complete
- Flow diagram included
- All checklists signed off

# Paper Processing

**Purpose**: Extract systematic data from individual academic papers to enable cross-study comparison and synthesis.

**When to use**: After defining research question and collecting papers to review.

---

## Step 1: Extract Paper Metadata

For each paper, systematically extract the following information:

### Basic Information
- Authors (last name, first initial)
- Year published
- Title
- Journal/Conference
- DOI or URL

### Study Design
- Research method (experimental, observational, meta-analysis, qualitative)
- Sample size (N=?)
- Population studied
- Independent/dependent variables

### Key Findings
- Main results (with effect sizes if reported)
- Statistical significance
- Practical significance

### Theoretical Framework
- What theory guides the research?
- What prior work does it build on?
- Who is cited as foundational?

### Limitations
- Sample limitations
- Methodological constraints
- Generalizability concerns

## Best Practices for Paper Processing

### Critical Reading
- Don't accept findings at face value
- Check sample sizes and effect sizes
- Note conflicts of interest
- Consider alternative explanations

### Efficiency Tips
- Process papers in batches (3-5 at a time)
- Use template consistently for all papers
- Extract data as you read (don't re-read later)
- Note unclear points immediately

### Quality Indicators
- Large sample sizes (context-dependent)
- Effect sizes reported (not just p-values)
- Clear methodology description
- Limitations acknowledged
- Pre-registered studies (when applicable)

---

## Common Extraction Challenges

### Challenge 1: Missing Information
**Problem**: Paper doesn't report effect sizes or full methodology.

**Solution**:
- Note what's missing in limitations
- Mark for follow-up if critical
- Consider excluding if too incomplete

### Challenge 2: Complex Statistics
**Problem**: Unfamiliar statistical methods.

**Solution**:
- Focus on practical interpretation
- Note the method used
- Seek clarification if needed for synthesis

### Challenge 3: Contradictory Findings
**Problem**: Paper contradicts earlier findings.

**Solution**:
- Note the contradiction explicitly
- Look for methodological differences
- This is valuable data for synthesis phase
